# 📊 Ciclo 15 Formativo em Análise de Dados — PretaLab

## ✨ Sobre o projeto
Este repositório reúne materiais, atividades e aprendizados desenvolvidos ao longo do Ciclo Formativo em Análise de Dados da PretaLab.

O programa tem como objetivo capacitar mulheres, especialmente negras e pessoas LGBTQIA+, para atuação na área de dados, promovendo inclusão, diversidade e desenvolvimento técnico.

## 🎯 Objetivo
Desenvolver habilidades práticas e teóricas em análise de dados, incluindo:

* Manipulação e tratamento de dados
* Análise exploratória (EDA)
* Visualização de dados
* Uso de ferramentas e bibliotecas essenciais
* Pensamento analítico aplicado a problemas reais


## 🛠️ Tecnologias e Ferramentas
Durante o ciclo, foram utilizadas principalmente:

* Python
* Pandas
* Matplotlib
* Seaborn
* SQL
* Google Colab

## 📊 Atividades Desenvolvidas
* Limpeza e tratamento de dados
* Análise exploratória de datasets
* Criação de gráficos e visualizações
* Resolução de problemas com dados reais
* Desenvolvimento de projetos práticos

## 📈 Exemplos de Análises

Alguns insights trabalhados durante o ciclo:

* Identificação de padrões em dados
* Comparação entre categorias (ex: gêneros, produtos, etc.)
* Análise de tendências
* Geração de métricas e indicadores

## 🌱 Aprendizados

Durante o ciclo, foram desenvolvidas habilidades como:

* Raciocínio analítico
* Organização de dados
* Comunicação de resultados
* Uso de ferramentas do mercado de dados

## 💜 Considerações finais

Este repositório representa não apenas um processo de aprendizado técnico, mas também um passo importante na construção de uma trajetória na área de dados, com foco em diversidade e inclusão.
